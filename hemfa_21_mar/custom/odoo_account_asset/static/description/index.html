Account Asset Management for Odoo Community Edition

Account Asset Manage and Asset Depreciations in Odoo

This app allows users to manage accounting assets and depreciation into Odoo. Your user can configure asset types/categories. Depreciation will be computed as per settings and journal entries for depreciation will be created using the wizard shown in the below screenshots.

Main Features:

  • Allow you to set asset type on product form which will allow you to auto-create an asset from the vendor bill.
  • The system will create an asset automatically if the vendor bill will be set with the Asset type/category so the validation of the vendor bill system will create an asset automatically.
  • Allow you to compute depreciation of assets for periods as configured.
  • There is a wizard which will create a depreciation entry for assets when you will run that wizard by selecting the date.
  • Account asset types menu for setting up types of assets.
  • The system adds assets list view and form view as per screenshots.
  • View assets analytic report with depreciation date & vendor as per the below screenshot.
  • View Bar Chart, Line Chart, and Pie Chart for assets analytic report.
  • For more details please watch the video and check the below screenshots.
  • This app is only compatible with the Odoo community edition.
