Price Checker Kiosk

User can either scan the barcode or type the barcode number using the keypad

Select settings->users->others->

Put tick mark on Access to Price Checker in Kiosk Mode then refresh your browser

Select ->Price Checker -> Kiosk Mode

when the user types(scan) the barcode number the corresponding details of the product are loaded below automatically

After every scanning text box will be cleared automatically so the user can check other barcode
