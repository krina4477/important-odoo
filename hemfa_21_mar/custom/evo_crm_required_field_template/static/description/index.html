CRM Required Field Template

This modules helps user in making certain fields of CRM mandatory. User can keep validation and make desired field mandatory in Lead/Opportunity. It also gives user flexibility to make these validation stage wise.

Key Features

User can configure validation for fields of lead/opportunity in CRM.

User will be able to configure mandatory lead/opportunity fields for each stage.

Is this app compatible with Odoo Enterprise as well as Community?

Yes, the app works perfectly well with Odoo Enterprise (On-premise and Odoo.SH) as well as Community. Odoo Online (Cloud) does not allow installation of third-party apps and hence this app cannot be installed on Odoo Online.
