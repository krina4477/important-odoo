Manage Your Fixed Assets

This module allows you to manage your fixed asset and keeps track of depreciation's, and sell or dispose them.

Overview

This module allows managing assets owned by a company or a person. Keeps track of depreciation, and creates corresponding journal entries. The module will enable you to generate monthly depreciation entries automatically or manually, ability to set value of depreciation already done in another system, sell or dispose of assets, and generate assets reports.

Key Features
Ability to create assets types.
Ability to create asset manually or automatically from vendor bill.
Ability to set the already depreciated value to be excluded from Odoo calculation, In case of migration from another software
Ability to sell and dispose assets in an easy way.
Ability to open assets from vendor bill, In case of asset creation from vendor bill.
Ability to create product and assign asset type for it, and the asset type will be selected automatically in vendor bill
