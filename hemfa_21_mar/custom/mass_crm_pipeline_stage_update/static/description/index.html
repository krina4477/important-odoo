Update Stage CRM Opportunity / Pipeline in Bulk

Mass Update CRM Records Stage

This app will allow your sales team user to update CRM pipelines / leads / opportunity from the list view in bulk for the same team. If the user tries to select record(s) from different sales teams on the list then the system will raise a warning.

Main Features:

  • Allow your sales team user to move the CRM pipeline / opportunity / leads stage in bulk from the list view of the CRM records for the same sales team.
  • System will only list stages on wizard which belong to that sales team or for stages where the team has not been set.
  • System will write a stage on selected records on CRM Lead / Opportunity / Pipeline records as shown.
  • For more details please see below screenshots and watch the video.
