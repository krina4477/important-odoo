Asset Maintenance in Odoo

Main Features:

  • New fields are added on the Maintenance Request Form of Odoo Standard with Buttons/Actions as per below screenshots.
  • Users can approve Maintenance request, maintenance complete, Create Requisition, Submit to Manager, Receive Request.
  • Added Fields for Employee, Department, Partner to the Maintenance Team form.
  • Manage Maintenance Diagnosis.
  • Manage Maintenance Activity.
  • Allow you to create material requisition from Maintenance requests.
  • Workflow on Maintenance request form as per below screenshots.
  • Cron job to manage Maintenance request preventive steps.
