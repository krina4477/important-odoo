Restrict CRM Lead Opportunities Delete

Description

This module is configured to restrict the delete option for CRM Lead Opportunities.

Configuration

After installing the Restrict CRM Lead Opportunities Delete application, the feature will be enabled.

Click on Settings - - > Users & Companies - - > Users.

Click on the Admin user.

Under Access Rights, we can see “Can Delete CRM Lead Opportunities” which is disabled.

This functionality is used to restrict the delete option for My Pipeline, My Activities and Leads.

Checking for Pipeline. Go to CRM - - > Pipeline. View the pipeline in List View.

Select all the opportunities in the pipeline and select the Action button, we can see that the delete option is disabled.

Click on any of the opportunities.

We can see that the delete option is disabled for individual opportunity as well.

For the same user, we are enabling the “Can Delete CRM Lead Opportunities” under access rights.

Go to CRM - - > Pipeline. View the pipeline in List View.

Select all the opportunities in the pipeline and select the Action button, we can see that the delete option is enabled.

Click on any of the opportunities.

.

We can see that the delete option is enabled for individual opportunity as well.
